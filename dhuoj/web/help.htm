Tutorial of the DHUOJ Client

  1. Download the necessary softwares at http://acm.dhu.edu.cn/dhuoj/download.htm.

  2. Launch the client with either of the following ways:

    • Click the link “here” on the page http://acm.dhu.edu.cn/dhuoj/index.htm.



      or

    • Run this command in you command line terminal:

      javaws http://acm.dhu.edu.cn/dhuoj/launch.jnlp

  3. Click the button "Run" at the warning dialog.



  4. Input you ID and password in the login dialog,



  5. Click the "ShowContest" button and select the contest that you want to join.



  6. The client window opens if login successfully. Choose the menu “Tool” → “SetEnvironment” to set your compiler paths.
